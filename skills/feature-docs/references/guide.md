# Feature Documentation Guide

## Propósito
Esta skill permite documentar funcionalidades del proyecto donde se coloque.

## Cómo funciona

1. Analiza el código existente
2. Identifica funcionalidades clave
3. Genera documentación estructurada

## Buenas prácticas

- Documentar el propósito de cada feature
- Incluir ejemplos de uso
- Mantener actualizada la documentación
